🌍 Community Aid Bridge

A web-based platform that connects NGOs, Volunteers, and Donors to make community support faster, easier, and more organized.

Developed as a Social Impact Hackathon Project using HTML, CSS, JavaScript, Firebase Firestore, Chart.js, and Font Awesome.

---

📌 Problem Statement

NGOs often struggle to find volunteers, donors, and essential resources at the right time. Likewise, many people are willing to help but don't know where their support is needed.

Community Aid Bridge bridges this gap by providing a single platform where NGOs can post their needs while volunteers and donors can easily register and contribute.

⚙️ Installation

1. Clone the repository

https://github.com/disha26tawar-cloud/community-aid-bridge

2. Open the project folder

cd community-aid-bridge

3. Configure Firebase

- Create a Firebase project.
- Enable Firestore Database.
- Copy your Firebase configuration into "firebase.js".

4. Run the project

Open "index.html" using Live Server or deploy using Firebase Hosting.
